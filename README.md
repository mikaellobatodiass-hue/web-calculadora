# 🧮 Calculadora Web

Uma calculadora simples, moderna e funcional desenvolvida com HTML, CSS e JavaScript.

## 🚀 Demonstração

📌 Interface com tema dark (azul escuro)  
📌 Operações básicas: soma, subtração, multiplicação e divisão  
📌 Layout moderno e responsivo  

---

## 🧠 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
